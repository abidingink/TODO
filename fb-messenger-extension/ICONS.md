# Icon Placeholder

Create simple icons for the extension:

1. **icon16.png** - 16x16 pixels
2. **icon48.png** - 48x48 pixels  
3. **icon128.png** - 128x128 pixels

Use any image editor or online tool to create these icons with:
- Blue background (#1877f2 - Facebook blue)
- White "@f" text in the center
- Simple, clean design

If you can't create icons, the extension will still work - Chrome will show a default icon.

**Quick online icon generator options:**
- https://favicon.io/
- https://www.canva.com/
- https://www.flaticon.com/ (search for "chat bot" icons)
